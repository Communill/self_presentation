# Проект Самопрезентации

Это веб-приложение на Django для демонстрации моих навыков и опыта.

## Описание

Проект включает две основные страницы:

1. **Главная страница** - визуальное представление с анимацией и контактной информацией
2. **Страница нейронной сети** - интерактивная демонстрация работы нейронной сети для распознавания цифр

## Технологии

- Python 3.14.2
- Django 5.2
- HTML, CSS, JavaScript
- Canvas API для визуализаций

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <url_репозитория>
```

2. Создайте и активируйте виртуальное окружение:
```bash
python -m venv venv
venv\Scripts\activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Выполните миграции:
```bash
python manage.py makemigrations
python manage.py migrate
```

5. Запустите сервер:
```bash
python manage.py runserver
```

6. Перейдите в браузере по адресу `http://127.0.0.1:8000`

## Переменные окружения

Создайте файл `.env` в корне проекта с следующим содержимым:
```
SECRET_KEY=ваш_секретный_ключ
DEBUG=True
ALLOWED_HOSTS=localhost,127.0.0.1
```
